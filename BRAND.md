# Supper Club Brand System

## Positioning

Supper Club is a sophisticated hosting platform for thoughtful gatherings. The product should feel warm, editorial, calm, and intentional: more hospitality notebook than party flyer.

Core feeling: "Let's host people well."

## Palette

| Token | Hex | Use |
| --- | --- | --- |
| Warm Ivory | `#FAF7F2` | Page background, primary app surface |
| Soft Stone | `#F1ECE5` | Secondary panels, quiet hover states |
| Clay | `#E8DED2` | Warm low-contrast surface tint |
| Deep Charcoal | `#2A2623` | Primary text, dark sections |
| Olive | `#68735F` | Primary CTA, icons, success accents |
| Sage | `#8AA17B` | Supporting accents and selected states |
| Muted Terracotta | `#B86A4F` | Sparse secondary emphasis and warnings |
| Honey | `#C9A66B` | Small highlight accents only |

## Typography

- Large headings use Playfair Display with Cormorant Garamond/Georgia fallbacks.
- Body text and UI use Inter with system sans fallbacks.
- Headings should feel editorial and measured. Body text should stay compact, readable, and practical.
- Eyebrows use uppercase Inter with generous tracking.

## UI Principles

- Prefer warm ivory and stone surfaces over bright white.
- Use fewer borders; when needed, use low-opacity charcoal or olive.
- Use subtle shadows only.
- Buttons are refined rounded rectangles, not bubbly pills.
- Primary buttons use olive or dark olive.
- Terracotta is an accent, not a dominant color.
- Avoid neon colors, confetti, loud gradients, emoji-heavy UI, and playful startup styling.

## Components

- Cards: `8px` radius, warm surface, subtle shadow, minimal border.
- Inputs: cream fill, olive focus ring, restrained shadow.
- Badges: small rounded rectangles, quiet fills.
- Empty states: calm botanical cue, no celebratory sparkle treatment.
- Event imagery: warm, real hospitality photography with a charcoal overlay for legible editorial type.
